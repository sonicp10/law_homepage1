# ⚖️ 법무법인 프리미엄 (law_homepage1) 프로젝트 지침

이 파일은 본 홈페이지 구축 프로젝트의 핵심 가치와 기술적 방향성을 정의하며, 모든 작업 시 최우선적으로 참조되어야 합니다.

## 1. 프로젝트 개요 (Overview)
- **목적**: 법무사 사무소 홍보 및 고객 유치를 위한 프리미엄 홈페이지 구축.
- **타겟**: 채무 문제 해결, 민사/등기 서비스가 필요한 20세 이상의 성인 고객.
- **슬로건**: "압도적 탕감률, 실패 없는 새로운 시작"

## 2. 디자인 원칙 (Design Principles)
- **핵심 컬러**: Deep Navy (`#0F172A`) & Gold (`#C5A059`) - 전문성과 신뢰감 강조.
- **UI 스타일**: 유리 모피즘(Glassmorphism), 마이크로 애니메이션, 반응형 레이아웃.
- **톤앤매너**: 무겁고 권위적이지만 친절하고 상세한 정보 제공.

## 3. 핵심 기술 스택 (Tech Stack)
- **Frontend**: React (Vite base), Vanilla CSS (index.css 중앙 관리).
- **Backend**: Node.js (Express), SQLite (상담 신청 DB).
- **SEO**: `react-helmet-async` (Meta tags), Semantic HTML5.
- **Icons**: `lucide-react`.

## 4. 콘텐츠 및 기능 트리 (Core Features)
- **Main**: 히어로 섹션, 성공 데이터 시각화, 서비스 카드, 자가진단 유도.
- **Sub Pages**: 법인 소개, 개인회생, 개인파산, 민사/형사, 등기/경매, 고객센터(FAQ).
- **Conversion**: 30초 자가진단 시스템, 실시간 상담 신청 현황(사회적 증거), 플로팅 퀵 메뉴.

## 5. 최우선 수행 가이드 (Critical Rules)
- **신뢰성(Trust)**: 대표 법무사 프로필과 법원 인가 결정문 등의 성공 사례 이미지를 항상 눈에 띄게 배치한다.
- **전환율(Conversion)**: 상담 신청의 허들을 낮추기 위해 '전화 한 통' 또는 '간편 자격 진단'을 유도하는 CTA를 모든 페이지에 포함한다.
- **SEO**: 모든 페이지는 고유한 Title과 Meta Description을 가지며, 검색 엔진 친화적인 시맨틱 태그 구조를 유지한다.
- **언어 원칙**: 모든 콘텐츠와 UI는 정중하고 신뢰감 있는 **한국어(한글)**를 사용한다.
