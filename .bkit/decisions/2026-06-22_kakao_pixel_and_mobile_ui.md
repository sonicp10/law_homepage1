# 2026-06-22: 모바일 텍스트 줄바꿈 개선 및 카카오 픽셀 연동

## 1. 개요
모바일 환경에서 히어로 섹션의 텍스트가 부자연스럽게 줄바꿈되는 현상을 해결하고, 카카오 비즈니스 광고 효율화를 위해 카카오 픽셀(Kakao Pixel) 방문 이벤트 스크립트를 홈페이지에 연동했습니다.

## 2. 작업 내역

### 1) 모바일 히어로 텍스트 줄바꿈(UI) 개선
- **문제점:** 메인 페이지 모바일 화면에서 "법무사 김형근 사무소가" 텍스트 중 "가" 한 글자만 다음 줄로 넘어가 가독성을 저해하는 문제가 있었습니다.
- **해결책:** 
  - `word-break: keep-all`, `overflow-wrap: break-word` CSS 속성을 적용하여 한국어 단어 단위로 자연스럽게 줄바꿈되도록 개선했습니다.
  - `<br />` 태그에 의존하던 강제 줄바꿈을 제거하고, 논리적인 문장 단위로 `<span className="block">`으로 감싸 구조적으로 줄바꿈이 일어나도록 리팩토링했습니다.
- **관련 파일:** `src/app/page.tsx`

### 2) 카카오 픽셀(Kakao Pixel) 연동
- **목적:** 홈페이지 방문자 데이터를 카카오 광고 시스템으로 전송하여 리타겟팅 등 마케팅 효율을 높이기 위함입니다.
- **구현 방식:**
  - `src/components/analytics/KakaoPixel.tsx` 컴포넌트를 신규 생성하여 카카오에서 제공하는 추적 스크립트를 `next/script`를 활용해 삽입했습니다.
  - 하드코딩 방지를 위해 픽셀의 고유 `Track ID`는 환경 변수(`NEXT_PUBLIC_KAKAO_PIXEL_ID`)로 관리하도록 설정했습니다.
  - 전역 레이아웃인 `src/app/layout.tsx`에 `KakaoPixel` 컴포넌트를 마운트하여 모든 페이지 로드 및 이동 시(SPA 라우팅 포함) 방문(PageView) 이벤트가 전송되도록 처리했습니다.
- **관련 파일:** 
  - `src/components/analytics/KakaoPixel.tsx` (신규)
  - `src/app/layout.tsx`
  - `.env.local` (로컬 개발 환경 변수 추가)

## 3. 배포 및 검증
- Vercel 프로덕션 환경 변수에 `NEXT_PUBLIC_KAKAO_PIXEL_ID` 값(`8252409972395820788`)을 등록하고 재배포를 완료했습니다.
- 카카오 픽셀 헬퍼(Chrome Extension)를 통해 라이브 도메인(`wooriro.co.kr`)에서 정상적으로 PageView 이벤트가 수집됨을 확인했습니다.
- 카카오 모먼트 관리자 센터에서도 픽셀 데이터가 "수집 중" 상태로 활성화된 것을 최종 확인했습니다.
