# 의사결정: 관리자 대시보드 실시간 관제 센터 개편 및 퀵 액션 모달/알림음 시스템 구축

## 컨텍스트 (Context)
기존 법무사 사무소의 관리자 대시보드는 단순히 통계 카드와 단순 리스트만 렌더링하고 있었습니다.
사무소의 실무 운영진(법무사 및 담당 직원)이 새로운 방문 상담이나 전화 상담, 빠른 상담 신청(리드)이 접수되었을 때 즉시 인지하지 못하고 수동으로 각 관리 페이지(`/admin/consultations`, `/admin/leads`)로 매번 이동해야만 상세 내역을 열어보고 진행 상태를 처리할 수 있는 오퍼레이션 비효율성이 존재했습니다.

## 결정된 솔루션 (Proposed Solution)
대시보드를 단순 지표판이 아닌 **"실시간 통합 관제 센터(Operations Control Center)"**로 전면 업그레이드하여, 대시보드 내에서 마우스 원클릭으로 모든 신규 상담을 즉시 확인하고 직접 처리할 수 있는 강력한 기능 강화 시스템을 구축했습니다.

*   **1. 대시보드 퀵 액션 상세 모달 (Quick Details & Instant Update Modal)**:
    - 대시보드의 "최근 빠른 상담", "최근 전화/방문 상담", "최근 게시판 문의"의 모든 행 아이템을 클릭 가능한 인터랙티브 요소로 전환했습니다.
    - 항목 클릭 시, 의뢰인의 상세한 신청 조건(부채 규모, 지역, 방문 예약 날짜 및 시간, 상담 세부 고충 내용)이 일목요연하게 표시되는 **상세 카드 모달**이 열립니다.
    - **즉각 상태 변경 단추**: 모달 내부에서 `대기중`, `연락완료`, `진행중`, `완료`, `취소` 상태 단추를 원클릭하면 백엔드 PATCH API를 즉시 호출하여 DB 상태를 업데이트하고, 대시보드 데이터를 실시간 갱신합니다.
    - **원클릭 다이얼/복사**: 전화번호 우측에 복사(Clipboard) 및 바로 전화걸기(`tel:`) 버튼을 연동하여 사무실 전화나 스마트폰 연동 시 즉시 다이얼링이 가능하도록 마감했습니다.

*   **2. 실시간 신규 상담 유입 감지 & 토스트/오디오 알림음 시스템**:
    - 대시보드 백그라운드 폴링(10초 주기) 시, 이전 데이터와 새로운 데이터를 ID 단위로 정밀 분석하여 신규 상담 유입 여부를 실시간으로 감지합니다.
    - **인앱 토스트 알림**: 새로운 상담이 감지되면 화면 우측 하단에 역동적이고 세련된 bounce 애니메이션 배너로 접수 토스트 알림을 즉시 띄웁니다.
    - **자체 오디오 신디사이징 차임벨 (Web Audio API)**:
      - 별도의 외부 오디오(.mp3) 에셋 파일 의존성 없이, HTML5 Web Audio API를 활용하여 C5-E5 음계로 이어지는 **청아하고 고급스러운 알림음**을 자체 합성음으로 재생합니다.
      - 현대 브라우저의 Auto-Play 정책을 준수하여, 사용자가 대시보드 우측 상단에서 `🔔 실시간 알림음 On/Off` 버튼으로 직접 알림을 제어할 수 있도록 세심하게 설계했습니다. (설정 값은 `localStorage`에 자동 보존됩니다.)

*   **3. 오늘의 방문 상담 현황 요약 위젯 (Today's Scheduled Visitors)**:
    - 한국 시간(UTC+9) 기준 당일 예정된 방문 예약 목록이 존재할 경우, 대시보드 최상단에 **"오늘 예정된 방문 상담 일정"** 브리핑 섹션이 보라색 펄싱(Pulsing) 인디케이터와 함께 노출됩니다.
    - 사무소 직원이 오늘 방문할 의뢰인 명단과 시간대를 출근하자마자 한눈에 인지할 수 있어 고객 만족도를 극대화할 수 있습니다.
