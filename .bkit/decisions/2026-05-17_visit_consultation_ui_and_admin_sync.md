# 2026-05-17 Visit Consultation UI and Admin Sync

## Context & Motivation
사용자가 방문 상담 페이지에서 전화번호를 너무 길게 입력하거나, 예약시간/거주지역 선택 창에 드롭다운 화살표(Chevron)가 없어 사용성이 떨어지는 UI 문제가 존재했습니다. 또한 사용자가 성공적으로 방문 예약을 제출해도, 관리자 페이지(어드민 대시보드 및 상세 목록)에서 예약 일시(날짜 및 시간)가 누락되어 관리자가 예약을 확인할 수 없는 심각한 데이터 흐름 단절 문제가 있었습니다. 이를 해결하고 사이트 사용성과 관리 기능을 완벽하게 동기화하기 위한 개선 작업을 진행했습니다.

## Decisions

### 1. 연락처 실시간 포매팅 및 입력 제한
- **적용 대상:** `src/app/qna/visit/page.tsx`, `src/app/qna/phone/page.tsx`
- **결정 사항:** `formatPhone` 유틸리티 함수를 연동하여 사용자가 번호를 입력할 때 자동으로 `010-XXXX-XXXX` 포맷으로 하이픈이 생성되게 하고, 최대 길이를 11자리(하이픈 포함 13글자)로 강제 제한했습니다.
- **이유:** 부정확한 전화번호 입력을 사전에 원천 차단하여 법률 상담 데이터의 무결성을 확보하기 위함입니다.

### 2. 셀렉트 박스 드롭다운 화살표 UI 개선
- **적용 대상:** `src/app/qna/visit/page.tsx` ('예약시간' 및 '거주지역')
- **결정 사항:** `appearance-none` 스타일로 브라우저 기본 화살표가 가려진 셀렉트 박스를 relative 래퍼로 감싸고 absolute 위치의 커스텀 SVG 화살표를 삽입했습니다.
- **이유:** 사용자가 직관적으로 선택 가능한 드롭다운 박스임을 인지하게 함으로써 전반적인 사이트의 시각적 완성도와 프리미엄 브랜드 아이덴티티를 강화합니다.

### 3. 어드민 대시보드 및 상세 관리 연동
- **적용 대상:** `src/app/api/admin/stats/route.ts`, `src/app/admin/consultations/page.tsx`, `src/app/admin/page.tsx`
- **결정 사항:** 
  - 최근 상담 신청 목록을 반환하는 API에서 누락된 `visitDate`, `visitTime` 필드를 select 쿼리에 포함했습니다.
  - 상담 관리 테이블에서 방문 상담의 경우 기존의 비어있던 업무분야 란 대신 보라색 배지 형태(`📅 YYYY-MM-DD HH:MM`)로 예약일시를 표시하고, 테이블 헤더도 "업무분야 / 예약일시"로 고도화했습니다.
  - 대시보드의 최근 활동 목록에서도 `미분류`로 뜨던 방문 예약을 실제 예약 날짜/시간으로 대체 렌더링했습니다.
- **이유:** 관리자가 예약 건에 대해 즉각적으로 언제 방문 예정인지를 직관적으로 한눈에 확인하여 빠른 피드백을 전달할 수 있도록 관리 편의성을 제고하기 위함입니다.

## Status
- **Status:** Approved & Merged
- **Implementation:** 프론트엔드/백엔드 최적화 완료, 로컬 빌드 성공.

## Consequences
- 방문 예약 신청의 무결성이 향상되고, 셀렉트 디자인의 미적 완성도가 대폭 올라갔습니다.
- 관리자가 예약일정을 한눈에 즉각 식별할 수 있어 원활한 법률 상담 스케줄링 운영이 보장됩니다.
