# 2026-06-13 Greeting Profile Sketch Update and Hosting Diagnosis

## Context
1. **인사말 프로필 이미지 업데이트**: 기존에 사용하던 대표 법무사의 사진/엠블럼 대신, 부드럽고 몽환적인 파스텔 톤의 사선 해칭(Hatching) 스케치 기법을 적용한 "업무에 집중하는 법무사" 이미지를 새롭게 생성하여 인사말 페이지에 반영하고자 했습니다.
2. **호스팅 중단 원인 진단**: 사용자가 커스텀 도메인(`wooriro.co.kr`)으로 사이트에 접속할 수 없는 "호스팅 중단" 현상을 겪었으며, 이에 대한 원인 파악과 해결 방안 도출이 필요했습니다.

## Decision & Actions

### 1. 인사말 스케치 이미지 적용
- AI 이미지 생성 도구를 활용하여, 사용자가 제시한 레퍼런스(밝은 노출, 빗살 무늬 해칭, 몽환적 분위기)를 완벽히 모사한 3단계의 스케치 이미지를 반복 생성했습니다.
- 최종적으로 핑크/블루 파스텔 톤이 미세하게 가미된 따뜻한 감성의 빈 사무실 내 업무 중인 법무사 스케치(`lawyer_working_sketch.png`)를 확정했습니다.
- `src/app/intro/greeting/page.tsx` 내부의 `<Image>` 경로를 업데이트하고 Git에 커밋/푸시하여 홈페이지에 실시간 반영했습니다.

### 2. 호스팅(접속 불가) 원인 진단 및 해결
- **초기 오진**: 과거 작업 내역(`auth.ts`의 `JWT_SECRET` 환경 변수 강제 검증 로직)과 가비아 DNS의 A 레코드 IP(`216.198.79.1`)를 보고, 환경 변수 누락 및 DNS IP 오설정(Vercel 기본 IP인 76.76.21.21이 아님)이 원인이라고 잘못 판단했습니다.
- **실제 원인 규명**: 
  - Vercel의 Environment Variables에는 이미 5월부터 필수 키들이 완벽히 등록되어 있었고, 로컬/원격 빌드도 정상적으로 성공했습니다.
  - 가비아 DNS의 A 레코드(`216.198.79.1`) 또한 Vercel에서 과거 해당 도메인에 할당해 준 유효한(Valid) 연결 IP였습니다.
  - **진짜 원인**은 Vercel 대시보드의 **[Domains] 설정 탭에서 `wooriro.co.kr` 도메인 자체가 통째로 삭제(제거)**되어 있었던 것이었습니다.
- **해결 완료**: Vercel Domains 메뉴에 도메인을 다시 추가(`Add`)하자마자 기존의 완벽했던 가비아 DNS 설정과 맞물려 즉시 파란색 체크(Valid Configuration)가 뜨며 호스팅이 정상 복구되었습니다.

## Consequences
- 인사말 페이지의 시각적 퀄리티가 사용자 요구사항에 맞게 예술적이고 따뜻한 톤으로 크게 향상되었습니다.
- 커스텀 도메인 연동이 복구되어 일반 대중이 다시 정상적으로 홈페이지에 접근할 수 있게 되었습니다.
- AI 어시스턴트는 Vercel의 할당 IP 다양성을 간과하고 일반적인 A 레코드(76.76.21.21)만을 고집했던 판단 착오를 인지하고, 향후 호스팅 장애 진단 시 Vercel 내부의 도메인 등록 여부 등 가장 직관적인 요소를 우선순위로 확인하도록 지침을 갱신했습니다.
